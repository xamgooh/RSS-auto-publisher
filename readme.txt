Automatically import RSS feeds and enhance content with OpenAI GPT-5.

== Description ==

RSS Auto Publisher is a powerful WordPress plugin that automatically imports content from RSS feeds and enhances it using OpenAI's GPT-5 model. 

= Features =

* Import content from multiple RSS feeds
* Enhance content using GPT-5 AI
* Translate content to 7 languages (Spanish, Italian, Portuguese, Norwegian, Swedish, Finnish, Estonian)
* Queue-based processing system
* Customizable import settings per feed
* Featured image extraction
* Cost tracking for API usage
* Rate limit handling

= Requirements =

* WordPress 5.0 or higher
* PHP 7.4 or higher
* OpenAI API key (GPT-5 access)

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rss-auto-publisher` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to RSS Publisher > Settings to configure your OpenAI API key
4. Add RSS feeds from RSS Publisher > Manage Feeds

== Configuration ==

1. Get your OpenAI API key from https://platform.openai.com/api-keys
2. Enter the API key in RSS Publisher > Settings
3. Configure default post settings
4. Add RSS feeds with desired enhancement options
